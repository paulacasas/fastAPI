para clonar el repositorio 
git clone "link"

para agregar el archivo
git add "nombre"
git add . (se sube todo)

crear un comentario para subir al repositorio
git commit -m "descripción de cambios"

este sirve para hacer dos pasos al tiempo el git add y git commit solo con -am
git commit -am "descripción"

subir al repositorio
git push

eliminar un archivo 
git rm nombre archivo

git commit - m "elimine el archivo "
git push

eliminar carpeta
git rm -r nombre carpeta

git checkout -- [nombre archivo]

ver estado del archivo 
git status

ver ramas 
git branch 

descargar cambio de repositorio
git pull

si modifico cualquier archivo
git add "nombre"
git commit -m "descripción mas reciente"
git push

forzar el push remplazar lo remoto con mi historial local
git push --force

git log 
ver los comentarios de cada commit 
nos da el autor, fecha y su movimiento

para volver a un cambio anterior 
git reset  --hard <commit>
git reset --hard origin /master (version de GitHub)
